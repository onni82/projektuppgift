# projektuppgift
Projektuppgift för Programmering 1 med C++
 Syfte:
 Förmåga att formulera och planera programmeringsuppgifter med pseudokod och diagramteknik.
 Förmåga att skriva, läsa, strukturera, analysera, dokumentera och kommentera programkod.
 Färdigheter i att skapa program med ett givet syfte och för en avsedd användare.
 Förmåga att finna, analysera, åtgärda och förhindra syntaxfel, programkörningsfel och programmeringslogiska fel.
 Centralt innehåll
1. Grundläggande programmering i ett eller flera programspråk varav minst ett av språken är textbaserat.
2. Strukturerat arbetssätt för problemlösning och programmering.
3. Grundläggande kontrollstrukturer, konstruktioner och datatyper.
4. Arbetsmetoder för förebyggande av programmeringsfel, testning, felsökning och rättning av kod.
5. Grundläggande datastrukturer och algoritmer.
6. Gränssnitt för interaktion mellan program och användare.
7. Normer och värden inom programmering, till exempel läsbarhet, dokumentation, testbarhet, rena gränssnitt och nyttan av standard.

 Skicka in källkodsfilen som du döpt till <ditt_namn>.cpp
